Session Highlights

In this session, we deeply explored React Performance Optimization techniques. Specifically, we discussed when to use useMemo versus useCallback to prevent unnecessary re-renders in large lists. Sarah also shared insights on navigating the transition from Mid-level to Senior Engineer, emphasizing the importance of "System Design" skills over pure coding speed.

const ExpensiveComponent = memo(({ data, onItemClick }) => {
  // Only re-calculates if data changes
  const processedData = useMemo(() => {
    return data.map(item => heavyCalculation(item));
  }, [data]);

  return (
    <div>
      {processedData.map(item => (
        <ListItem key={item.id} onClick={onItemClick} />
      ))}
    </div>
  );
});
